MenuBot App

Product Goal: To enhance the dining experience by providing an interactive real-time digital menu that empowers users with detailed dish information, personalized recommendations, and seamless ordering, improving satisfaction, transparency, and efficiency for both diners and restaurants.

App Description

MenuBot is a real-time digital menu app that enhances dining experiences by offering detailed dish descriptions, allergen info, and personalized AI recommendations. The app integrates with POS systems, supports real-time updates, and provides multilingual support. Additionally, it includes customer feedback features, reducing the need for printed menus and promoting a smarter, sustainable dining experience.

Key Features & Functionalities

  • Interactive real-time menu with detailed dish descriptions and allergen information.
  • AI-powered personalized recommendations.
  • POS system integration and real-time updates.
  • Multilingual support and customer feedback functionality.
  • Eco-friendly approach reducing printed menu usage.

Triggers, Animations, and Variables

Logo Hover Effect: On hover, the logo scales up to increase visibility.

Food Ordering Mode: Toggle between Delivery and Pick-Up modes with a smooth sliding animation.

Add to Cart Interaction: Adding an item triggers a popup message that fades in and out.

Variables: Track the quantity of items (e.g., CarbonaraCount, VongoleCount) with conditional logic to prevent negative values.

Cat Breeds App

Objective: To create a user-friendly application that displays detailed information about various cat breeds in a visually appealing and dynamic format.

Purpose: The app is designed for cat enthusiasts who want to explore and learn about different breeds. It provides an engaging platform that combines information, usability, and design to create an enjoyable user experience.

Features

This app offers three main features:

  • Displays a list of cat breeds with key details such as breed name, origin, origin location, and coat type.
  • Includes a dynamic search bar that allows users to filter the breeds in real-time by typing keywords related to the breed name, origin, or coat type.
  • Features a custom cat logo for branding and utilizes dynamic colors and coat type icons for enhanced visual engagement.

How It Works

When the app starts, it fetches cat breed data from an online API and displays it using React Native's FlatList, which ensures efficient rendering of the list. The search bar dynamically listens for user input and updates the list in real-time by filtering the data.

Additionally, the app includes sorting functionality that allows users to organize the list alphabetically, by origin, or by coat type. Each breed is displayed in a card format with dynamic background colors and coat type icons, making it visually distinctive and easy to navigate. The responsive design is achieved using Flexbox and styled with React Native's StyleSheet.

  • Efficient rendering using React Native's FlatList for large data sets.
  • Dynamic filtering and sorting functionalities for a seamless user experience.

  • Responsive layout powered by Flexbox and styled using React Native's StyleSheet.
